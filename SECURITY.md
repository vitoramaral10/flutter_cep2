# Security Policy

## Reporting a Vulnerability

If you find a security vulnerability, **please do not open a public issue**.

Email **vitor.amaral10@gmail.com** with:

- A description of the vulnerability
- Steps to reproduce (minimal code sample if possible)
- Affected versions
- Potential impact

You can expect an acknowledgement within 48 hours and an update on the fix timeline shortly after.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |
| 0.x     | No        |
